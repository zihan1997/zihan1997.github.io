    Basics

    Deployment

    method 1:

    helm install [yaml] [name] [folder of chart]

    name: service name to be used to run

    method 2:

    kbectl apply -f

    Scale

    kubectl scale --replicas=0 deployment/app

    app: the name of service

    Deletion

    1. delete deployment

    kubectl delete deployment [name]

    2. delete service

    kubectl delete service [name]
